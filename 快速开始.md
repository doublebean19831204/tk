# 快速开始指南

## 📁 文件清单

```
题库/
├── index.html          ← 打开这个文件启动题库
├── questions.json      ← 编辑这个文件添加题目
├── 验证.py            ← 验证JSON格式是否正确
├── 转换指南.md        ← 详细的转换步骤
└── README.md          ← 完整的使用说明
```

---

## 🚀 三步使用

### 步骤1：准备题目
从你的Word题库中复制题目内容（至少复制几题看看格式）

### 步骤2：编辑 questions.json

**方法A：用记事本**
1. 右键点击 `questions.json`
2. 选择"打开方式" → "记事本"
3. 按照 `转换指南.md` 中的格式填入题目
4. 保存（Ctrl+S）

**方法B：用VS Code（推荐）**
1. 安装 VS Code（免费）
2. 用 VS Code 打开 `questions.json`
3. VS Code 会自动检查JSON格式
4. 边编辑边看到格式提示

### 步骤3：验证和测试

**验证格式：**
```bash
python 验证.py
```
或在文件夹中按住 Shift+右键，选择"在此处打开命令窗口"，输入上面的命令

**打开题库：**
- 双击 `index.html` 
- 或右键 → "打开方式" → 浏览器

## 📝 JSON 格式速查

### 完整示例
```json
{
  "水利工程安全": [
    {
      "question": "大坝的防渗指标应定期检测吗？",
      "options": ["A. 应该", "B. 不应该"],
      "correct": 0,
      "explanation": "大坝防渗是重要指标，必须定期检测。"
    },
    {
      "question": "常见的坝型是什么？",
      "options": ["A. 重力坝", "B. 拱坝", "C. 土坝", "D. 混合坝"],
      "correct": 2,
      "explanation": "土坝是最常见的坝型。"
    }
  ],
  "安全生产": [
    {
      "question": "主要负责人对安全生产全面负责。",
      "options": ["A. 正确", "B. 错误"],
      "correct": 0,
      "explanation": "这是安全生产法的基本规定。"
    }
  ]
}
```

### 关键要点
- `"分类名称"` - 可以改成你想要的名字
- `"question"` - 题目内容
- `"options"` - 答案选项（可以是2个、3个或4个）
- `"correct"` - 正确答案的索引：0=第一项，1=第二项，2=第三项，3=第四项
- `"explanation"` - 答案解释

## ⚠️ 常见错误

| 错误 | 原因 | 修复 |
|------|------|------|
| 浏览器打不开题库 | JSON格式错误 | 运行 `验证.py` 查看具体错误 |
| 题目显示不完整 | 缺少逗号或引号 | 检查JSON语法 |
| 答案不对 | correct值设置错误 | 确保correct值正确（0-3） |

---

## 💡 使用建议

1. **先测试**：先只添加3-5道题目，验证格式无误后再大量添加
2. **使用VS Code**：有自动错误检查和代码提示，效率更高
3. **保存前验证**：每次编辑后都运行 `验证.py` 检查
4. **备份**：定期复制备份 `questions.json`
