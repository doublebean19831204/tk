# 题库系统使用指南

## 🚀 快速开始

1. **打开题库**  
   双击 `index.html` 打开浏览器，开始做题

2. **编辑题目**  
   用记事本或代码编辑器打开 `questions.json`，按照格式添加或修改题目

## 📝 JSON 格式说明

```json
{
  "分类名称": [
    {
      "question": "题目文本",
      "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
      "correct": 0,
      "explanation": "答案解释"
    }
  ]
}
```

### 字段说明：
- **分类名称**: 题库分类（如"安全生产基础"）
- **question**: 题目内容
- **options**: 选项数组（可以2个、3个、4个选项）
- **correct**: 正确答案索引（0=第一项，1=第二项...）
- **explanation**: 答案解释说明

## 📋 示例格式

### 单选题
```json
{
  "question": "以下哪个不是安全生产的三个要素？",
  "options": ["A. 人", "B. 机器", "C. 物理环境", "D. 数字"],
  "correct": 3,
  "explanation": "安全生产三要素是人、机器、环境。数字不是要素。"
}
```

### 判断题
```json
{
  "question": "安全生产是第一位的。",
  "options": ["A. 正确", "B. 错误"],
  "correct": 0,
  "explanation": "安全生产确实是企业的首要任务。"
}
```

## 💡 使用技巧

1. **添加新分类**：在 JSON 中添加新的分类名称和题目即可
2. **修改题目**：直接编辑 JSON 文件后，刷新浏览器即可看到更新
3. **导入大量题目**：
   - 从 Word 复制题目
   - 粘贴到 JSON 格式中
   - 保存后刷新浏览器

## 🎯 功能特点

✅ 多分类支持 - 可以组织不同主题的题目  
✅ 即时反馈 - 选择答案后立即显示对错  
✅ 详细解释 - 每题都有答案解释  
✅ 进度追踪 - 显示学习进度和成绩  
✅ 完全离线 - 无需网络，本地运行  

## 🔄 更新步骤

1. 打开 `questions.json`
2. 添加或修改题目（注意 JSON 语法）
3. 保存文件
4. 刷新浏览器（F5）
5. 开始做题

## ⚠️ 注意事项

- JSON 格式必须正确（逗号、括号要匹配）
- 题目中的特殊字符（如引号）需要转义
- `correct` 的值必须是有效的选项索引

## 📞 遇到问题？

如果 JSON 格式有问题，浏览器控制台会提示错误。  
按 F12 打开开发者工具，查看 Console 标签的错误信息。
