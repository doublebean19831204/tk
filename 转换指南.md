# JSON 转换步骤指南

## 🎯 三步完成转换

### 第1步：打开文件

1. 打开 `questions.json`（用记事本或VS Code）
2. 打开 Word 题库文档
3. 并排放在屏幕上

### 第2步：理解JSON结构

```
{
  "分类名称": [        ← 题目分类（可改）
    {
      "question": "题目",
      "options": ["A. ...", "B. ..."],
      "correct": 0,     ← 0=第一个选项, 1=第二个...
      "explanation": "解释"
    }
  ]
}
```

### 第3步：复制题目

**判断题示例：**

Word 中的题目：
```
1. 安全生产是第一位的。
   A.正确  B.错误
   答案：A
```

转换为 JSON：
```json
{
  "question": "安全生产是第一位的。",
  "options": ["A. 正确", "B. 错误"],
  "correct": 0,
  "explanation": "安全生产是企业的首要任务。"
}
```

---

**单选题示例：**

Word 中的题目：
```
2. 生产经营单位的主要负责人是谁？
   A.董事长  B.总经理  C.主要负责人  D.安全员
   答案：C
```

转换为 JSON：
```json
{
  "question": "生产经营单位的主要负责人是谁？",
  "options": ["A. 董事长", "B. 总经理", "C. 主要负责人", "D. 安全员"],
  "correct": 2,
  "explanation": "根据安全生产法规定，生产经营单位的主要负责人是主要负责人。"
}
```

## 📋 快速复制模板

需要多道题目？复制这个模板：

```json
{
  "水利工程安全生产": [
    {
      "question": "题目1",
      "options": ["A. ", "B. ", "C. ", "D. "],
      "correct": 0,
      "explanation": "解释1"
    },
    {
      "question": "题目2",
      "options": ["A. ", "B. ", "C. ", "D. "],
      "correct": 0,
      "explanation": "解释2"
    },
    {
      "question": "题目3",
      "options": ["A. ", "B. ", "C. ", "D. "],
      "correct": 0,
      "explanation": "解释3"
    }
  ]
}
```

然后填入你的题目即可！

## ✅ 检查清单

转换时检查：

- [ ] 每个题目都有逗号分隔（最后一个除外）
- [ ] 双引号 `"` 完整配对
- [ ] 花括号 `{` `}` 完整配对
- [ ] 方括号 `[` `]` 完整配对
- [ ] `correct` 值是正确选项的索引（0, 1, 2 或 3）
- [ ] 所有特殊字符转义正确

## 🔍 JSON 验证

转换完成后：

1. 保存 `questions.json`
2. 在浏览器打开 `index.html`
3. 按 F12 打开开发者工具
4. 查看 Console 标签
5. 如果有红色错误，说明 JSON 格式有问题

## 💡 常见问题

**问：题目中包含双引号怎么办？**
```json
// 错误 ❌
"question": "某人说"安全第一""

// 正确 ✅
"question": "某人说\"安全第一\""
```

**问：题目中包含换行怎么办？**
```json
// 错误 ❌
"question": "题目第一行
题目第二行"

// 正确 ✅
"question": "题目第一行 题目第二行"
// 或用 \n 表示换行
"question": "题目第一行\n题目第二行"
```

**问：如何添加多个分类？**
```json
{
  "第一个分类": [ { ... } ],
  "第二个分类": [ { ... } ],
  "第三个分类": [ { ... } ]
}
```
