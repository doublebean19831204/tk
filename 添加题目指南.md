## 📚 如何添加完整的Word题目

### 方法1：手动复制粘贴（推荐新手）

**第1步：打开 questions.json**
- 用记事本或VS Code打开 `questions.json`

**第2步：复制Word题目**

从Word中复制题目内容后，按照以下格式添加：

#### 单选题示例
```json
{
  "question": "你的题目文本",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "correct": 2,
  "explanation": "答案解释"
}
```
- `"correct": 2` 表示答案是第3个选项（从0开始计数）

#### 多选题示例
```json
{
  "question": "你的题目文本",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4", "E. 选项5"],
  "correct": [0, 2],
  "explanation": "答案解释"
}
```
- `"correct": [0, 2]` 表示答案是A、C两项

#### 判断题示例
```json
{
  "question": "你的题目文本",
  "options": ["A. 正确", "B. 错误"],
  "correct": 0,
  "explanation": "答案解释"
}
```
- `"correct": 0` 表示正确，`"correct": 1` 表示错误

**第3步：保存并刷新**
1. 保存 `questions.json`
2. 在浏览器中刷新 F5
3. 新题目就会出现

---

### 方法2：批量导入（推荐大量题目）

**第1步：导出Word题目为文本**
1. 在Word中全选题库（Ctrl+A）
2. 复制（Ctrl+C）
3. 新建记事本，粘贴（Ctrl+V）
4. 保存为 `题库.txt`

**第2步：运行转换工具**
```bash
# 在题库文件夹打开命令窗口，运行：
python 题库转换器.py 题库.txt questions.json "水利工程安全"
```

**第3步：刷新浏览器**

## ✨ 功能说明

### 选择答案前
- ❌ 答案和解释都是隐藏的
- 只显示题目和选项

### 点击"确定"后
- ✅ 立即显示对错
- 显示正确答案（高亮为绿色）
- 如果错误，错误选择高亮为红色
- 显示详细的答案解释

### 答题完成
- 显示总分和正确率
- 显示各分类的得分统计
- 可以重新开始

## 🎯 建议步骤

1. **先小批量测试** - 添加5-10道题目，确保格式无误
2. **逐步添加** - 每次添加一批题目后刷新浏览器验证
3. **检查JSON格式** - 如果出现问题，运行 `python 验证.py` 检查
4. **完善解释** - 保证每道题都有清晰的答案解释

---

## ⚠️ 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 题库打不开 | JSON格式错误 | 检查逗号、引号、括号 |
| 部分题目不显示 | 分类名字与问题.json中不一致 | 确保分类名字拼写正确 |
| 答案判断错误 | correct值不对 | 检查correct的值是否正确 |
| 多选题只能选一个 | 题目类型设置错误 | 确保correct是数组[0, 1] |
